Hi, I’ve been working on **Horizon** — a desktop framework for Linux built completely from scratch, without GTK or Qt.

On top of that, I built **Austral OS**, a minimal Debian-based system where the entire user interface is powered by Horizon.

---

### Main ideas

- Full control over the UI stack
- Wayland-native
- Consistent system-wide design
- Inspired by classic macOS aesthetics (Mavericks / Leopard era)

---

### Current status

It’s currently in **alpha**:

- There are bugs and incomplete features
- Some parts (like settings and clipboard) are still being refined

---

### Included applications (in active development)

- Launchpad
- File manager
- Terminal
- Text editor
- Image viewer
- Document viewer
- Downloader
- Installer
- Session greeter
- Partition tool
- Basic web browser (experimental)
- Preferences (system settings)
- Desktop shell (panel + dock)
